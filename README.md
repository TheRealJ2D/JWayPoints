# JWaypoints

A feature-rich Minecraft waypoint plugin with **3D floating arrow navigation** and extensive customization options.

## ✨ Key Features

- **🎯 3D Arrow Guidance** - Follow beautiful floating arrows made of blocks that point toward your destination
- **🎨 Customizable Designs** - Choose from multiple arrow styles (Standard, Diamond, Gold, Emerald, Redstone) with glowing effects and particle trails
- **📊 Distance Display** - Track your progress using the XP bar as a distance meter
- **🎵 Arrival Effects** - Celebrate reaching your destination with custom sounds and particle effects
- **📱 Modern GUI** - Easy-to-use graphical interface for managing all your waypoints
- **👥 Waypoint Sharing** - Share your favorite locations with other players
- **✏️ In-Place Editing** - Rename waypoints and update coordinates without recreating them
- **🧭 Smart Guidance** - Automatically switches between arrow and compass guidance when arrows are blocked
- **⚙️ Highly Configurable** - Extensive configuration options for server administrators

## 🎮 Commands

- `/waypoint set <name>` - Create a waypoint at your location
- `/waypoint list` - Open the waypoint management GUI  
- `/waypoint activate <name>` - Start following a waypoint
- `/waypoint design <name>` - Change your arrow design
- `/waypoint sound <name>` - Customize arrival sounds
- `/waypoint share <name> <player>` - Share waypoints with friends

**Perfect for survival servers, creative builds, and any server where players need to navigate large worlds efficiently!**
